Sobre Smart Crowd

Smart Crowd es un sistema embebido para el control de aforo de determinados espacios. Nuestro proyecto desarrolla con disintas tecnologías, tales como la placa de desarrollo RaspberryPI4, lenguaje Python, algoritmo de YOLO, Google Colab para el entrenamiento de un modelo propio, Framework de Django y Microsoft Azure para el alojamiento de la página y para el almacenamiento y transferencia de datos en una base de datos de PostgreSQL.
